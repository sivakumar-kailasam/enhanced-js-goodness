## enhanced-js-goodnes
My extension of https://github.com/jelera/vim-javascript-syntax to have more goodness

### Requirements
- Vim 7.4+ (haven't tested on other versions)
- [Vim JavaScript plugin](https://github.com/jelera/vim-javascript-syntax)

### Contains
- Ember.JS functions 
  - refactoring
- ES6 highlighting
  - template strings
